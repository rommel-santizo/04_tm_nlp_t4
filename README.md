Tarea 4 - Topics Trip Advisor
==============================

Esta tarea identifica 5 Topics en un set de datos de Trip Advisor
Y genera un modelo para predecir a partir un input a que Topic pertenece
___
Ejecución:
------------
1. Leer y preparar datos:  **/src/data/prepare_dataset_tripad.py**
2. Generar el modelo:      **/src/models/train_model_tripad.py**
3. Visualizar los Topics:  **/src/visualization/visualize_tripad.py**
4. Predecir con el modelo: **/src/models/predict_model_tripad.py**
5. Test con un input utiliza Predict: **/src/models/test_tripad.py**
___
Denominación de Topicos
------------
1. El lugar Antigua y las cosas positivas
2. El hotel
3. Actividades para hacer
4. Descanso
5. Lugares
